# KCountries

## Usage

1. Add Jitpack to your root build.gradle
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
2. Add the dependency
```
dependencies {
	        implementation 'com.github.Freezor:KCountries:Tag'
	}
```
